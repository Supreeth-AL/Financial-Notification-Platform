# Analytics Pipeline

## Purpose

Provide real-time notification metrics for dashboards.

---

## Metrics

- Total Notifications
- Successful Deliveries
- Failed Deliveries
- Email Count
- SMS Count
- WhatsApp Count
- Push Count
- In-App Count
